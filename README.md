## Contributing

Start prompts with `_` so they're easy to `@` reference in tools like [Cursor](https://www.cursor.com/).

While ideally you're getting reliable results from a prompt prior to adding it to this repository, sometimes it's nice to iterate across various domains. Use the `beta` folder for these prompts.
